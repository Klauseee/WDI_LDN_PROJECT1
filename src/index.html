Learn2Code: Escape the Matrix

Moves:0

Instructions

Your friend is trapped inside the matrix, pass him commands so that he can make it to the phone box and escape. Use the input boxes on the right to enter your commands. The commands available to you are:

  • move forward - "f"
  • turn right - "r"
  • turn left - "l"
  • flick switch - "fs"

Commands are not case sensitive but will return an error if spaces are present

Try and clear each stage in as few moves as possible. The moves will not be reset by hitting the reset button!
